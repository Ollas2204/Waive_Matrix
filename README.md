# Waive Decision Tree

Tool interaktif untuk menentukan eligibilitas waive berdasarkan Waive Matrix Criteria.

## Deploy ke GitHub Pages

### 1. Upload ke GitHub
- Buat repository baru di GitHub dengan nama `waive-decision-tree`
- Upload semua file ini ke repository tersebut

### 2. Aktifkan GitHub Pages
- Buka repo → **Settings** → **Pages**
- Di bagian **Source**, pilih **GitHub Actions**
- Simpan

### 3. Selesai!
Setiap kali kamu push ke branch `main`, app otomatis ter-deploy ke:
```
https://<username-github-kamu>.github.io/waive-decision-tree/
```

## Jalankan Lokal (opsional)
```bash
npm install
npm run dev
```
